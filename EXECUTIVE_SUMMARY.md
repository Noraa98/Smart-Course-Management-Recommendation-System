?? TECHNICAL LESSONS LEARNED
???????????????????????????????????????????????????????????????

1. CSS Specificity Matters
   ? Use proper CSS classes, not !important

2. Z-index Can Be Simple
   ? Don't use z-index: 100, use z-index: 10
   ? Simpler stacking context = fewer conflicts

3. Flex Layout Edge Cases
   ? d-flex ? inline-flex
   ? Different use cases, different behaviors

4. Role Attributes Are Important
   ? <a role="button"> is not just for CSS
   ? Affects browser behavior and accessibility

5. Bootstrap Class Selection
   ? Choose classes based on use case
   ? d-flex is for layouts, not button groups
